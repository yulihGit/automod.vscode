# AutoMod README

AutoMod language support for VS Code.

(Primary for internal use. Developed during master's thesis)

## Features
* Syntax highlighting
* Snippets
* Auto-completion (AutoMod reserved keywords)
* Generator scripts (Snippets, keywords)
* Comment blocks
* CodeMap (mapper)
