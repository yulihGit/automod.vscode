# AutoMod README

AutoMod language support for VS Code

## Features
* Syntax highlighting
* Snippets
* Auto-completion (AutoMod reserved keywords)
* Generator scripts (Snippets, keywords)
* Comment blocks
* CodeMap (mapper)
